<h3>Web3:</h3>

* **WAGMI**          *#Connect wallet*
* **Ethers.js**      *#Create contract, send transactions etc*

<h3>Frontend:</h3>

* **React**          
* **Tailwindcss**    *#CSS framework for Classnames*
* **PostCSS**        *#Add js into CSS*

<h3>Helpers:</h3>
